## 2. Anatomy of a diagram

A source has three kinds of statements: **node** declarations, **edges**, and
**style** blocks.

<CodeBlock
  code={`node Planner {
  type: agent
  on: true
  status: running
}

node Researcher {
  type: subagent
  parent: Planner
  on: false
}

Planner -> Researcher

style node {
  background: node.on ? chili : node.parent.on ? bamboo : cream
  border: node.selected ? chili : border
  pulse: node.status == "running"
}`}
/>

That renders to a live diagram:

```mantou title="Hello, Mantou" height=340
node Planner {
  type: agent
  on: true
  status: running
}

node Researcher {
  type: subagent
  parent: Planner
  on: false
}

Planner -> Researcher

style node {
  background: node.on ? chili : node.parent.on ? bamboo : cream
  border: node.selected ? chili : border
  pulse: node.status == "running"
}
```
