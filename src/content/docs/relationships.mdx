## Accessors

- <Code>node.parent</Code> — the explicit parent reference
- <Code>node.children</Code> — direct children
- <Code>node.ancestors</Code> — parent chain to the root
- <Code>node.descendants</Code> — everything beneath
- <Code>node.neighbors</Code> — nodes connected by an edge
- <Code>node.incoming</Code> / <Code>node.outgoing</Code> — edges
